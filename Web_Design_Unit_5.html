Web Accessibility

What is Web Accessibility

Web Accessibility is what helps navigating and creating wibsites easier for all different people. These ways of making it easier to browse through different websites helps cater to people of old age or different diseases. The WCAG (Web Content Accesability Guide) is a guide on the internet about web accessibility made by the W3C. For more information click the link below.

https://www.w3.org/WAI/standards-guidelines/wcag/

Section 508 Compliance is a federal law that requires all federal agencies to ensure that their website is accessible to all people. For more information on Section 508, see the link below:

https://www.section508.gov/manage/laws-and-policies

Assistive technology used to browse the Internet

Below are different types of technology to help people with disabilities browse the Internet

1. JAWS is a screen reader that allows for those suffering from vision loss to be able to understand the content on a website. Is it avaliable for download on Windows 10.
2. MAGic is a tool that helps people with vison loss by allowing the user to zoom in on certain parts of their screen, displaying what is shown at a magnified state.
3. NaturalReader is a downloadable text-to-speech desktop software for personal use. This easy-to-use software with natural-sounding voices can read to you any text such as Microsoft Word files, webpages, PDF files, and E-mails.

Simulations for Web Accesibility

Online simulations are what help people see what's it like to have a condition that would affect your online experience.

*Below are a list of different simulations to show what it's like for people with different conditions:
1. Colorblind Web Page Filter - online web filter that allows for a person to see the colors on a website as a colorblind individual would.
2. Color Labrotory - emulates how individuals with colorblindess see the world.
